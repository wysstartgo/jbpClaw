# 商品库、资产共享与 DPA联机 Smoke Prompts

## 使用方式

- 先把本域 `tool-range.json` 装进 MCP 配置。
- 再用下面这些问题做最小联机验证。
- 预期目标是验证路由、只读发现、写前确认、结果摘要四件事。

## Smoke Prompt 1

- 提问：帮我查看商品库、DPA 商品和共享关系的整体状态。
- 验收点：
  - 模型优先调用本域只读工具
  - 模型不会越域乱调不相关工具
  - 若命中写操作或高风险工具，会先确认
  - 输出包含结果摘要和下一步建议
