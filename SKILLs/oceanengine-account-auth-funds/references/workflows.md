# 账户、鉴权与资金工作流蓝图

## 账户体检

- 目标：快速了解一个广告主账户的基础状态和潜在风险。
- 步骤：
  - 读取广告主基础信息、公开信息和预算配置。
  - 读取账户余额、流水和共享钱包相关状态。
  - 读取资质与安全积分信息，整理异常点。
- 完成标志：能给出账户摘要、异常项和后续动作建议。

## 鉴权前置检查

- 目标：判断当前 token 与账户关系是否足以支撑后续业务调用。
- 步骤：
  - 确认当前 token 类型和用户身份。
  - 读取目标广告主/钱包/资质是否可访问。
  - 输出缺失权限、可继续动作和阻塞点。
- 完成标志：能明确说出哪些接口可调、哪些仍缺前置。
