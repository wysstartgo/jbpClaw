# 小说漫画分镜 Contract

每个漫画页建议使用以下结构，便于后续生成图片和公众号正文。

```yaml
---
page: 1
title: "师兄弟重逢"
chapterId: "jianlai-chapter-0719"
lineStart: 126550
lineEnd: 126589
style: "wuxia"
layout: "cinematic"
aspect: "3:4"
---
```

## 页面内容

- **剧情目的**：这一页推进什么信息。
- **情绪关键词**：孤独、重逢、决裂、释然、问剑、守护等。
- **出场人物**：人物名和别名要与图谱别名一致。
- **场景**：地点、时间、天气、光线。
- **分格设计**：
  1. 远景：建立环境。
  2. 中景：人物动作。
  3. 近景：表情或关键物件。
  4. 收束格：情绪落点或反转。
- **画面提示词**：适合直接送入图像生成器。
- **正文解说**：公众号中放在图片下方的短文案。
- **证据锚点**：`chapterId + lineStart + lineEnd`。

## 质量标准

- 每页只讲一个剧情动作或一个关系变化。
- 人物外观不能在不同页大幅漂移。
- 画面不要塞满文字；公众号正文负责解释。
- 名场面可以用 splash page，但不要所有页都 splash。
- 战斗页优先表现势、空间和后果，不堆砌招式名。
